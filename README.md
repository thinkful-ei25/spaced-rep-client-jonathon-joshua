# Lesperanto

Lesperanto is a spaced repetition language learning app for Esperanto

![Category Select](./public/images/loginsnap.png?raw=true 'Title')

## Live App

[Lesperanto](https://esperanto-client.herokuapp.com/)

## Features

- Learn Esperanto
  - Animals
  - Food
  - Places
  - Technology
  - Locations
  - Directions
  - Phrases
